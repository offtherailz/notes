# Notes on perldoc

## Basics

+ `perldoc perl` for a very general overview
+ `perldoc perltoc` for a table of contents of all the documentation
+ `perldoc perllocal` to see what extra modules are installed
+ `perldoc perlmodlib` to see Perl's core modules

## Key uses for perldoc

+ `perldoc -f <query>` look up information on a specific function
+ `perldoc -q <query>` search the FAQ for something

## Good tricks

+ `perldoc -l <module name>` find the install path of a module
+ `perldoc -m <module name>` view the source of a module
+ `perldoc -q installed` techniques to view all the modules installed on a machine
